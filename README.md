### [개인] 실시간 경매 시스템

일정 : 25.12.08 ~ 개발중

기술 스택: Java, SpringBoot, JPA, MySQL, Redis, WebSocket

서비스: 상품 경매를 시작하면 실시간 입찰 금액을 갱신하고 낙찰가로 결제하는 서비스

기술 스택 : Spring Security, JPA, QueryDSL,  AWS EC2 S3 RDS, Nginx, Docker, Github Actions, SSE

개인 포트폴리오로 실시간 경매 시스템을 만들고 있습니다. 실시간으로 입찰 금액을 갱신하고(웹소켓+redis) 레디스에 모인 대용량 입찰 기록을 스케줄러로 db에 한번에 저장하여 개선했습니다. 이를 단위테스트로 검증할 예정입니다. 

추가할 구현 계획 

- 입찰 가격 기록 조회에서, 성능을 높이기 위해 인덱스 사용 → 복합인덱스(auctiton_id, price)
- 결제 기능에서, 낙관적 락을 사용해 재고가 음수가 되지 않도록 차단 → 재고가 1개일때 B가 주문중에 A가 결제완료하면 B의 주문은 실패
- 선착순 한정 수량 이벤트 (3순위)
